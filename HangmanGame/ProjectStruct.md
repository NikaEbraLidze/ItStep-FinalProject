1.  ­ЪЈЌ№ИЈ HangmanGame.DataAccess (Data Tier)
    рЃћрЃА рЃцрЃћрЃюрЃљ рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃерЃћрЃюрЃљрЃ«рЃЋрЃљрЃАрЃљ рЃЊрЃљ рЃгрЃљрЃЏрЃЮрЃдрЃћрЃЉрЃљрЃќрЃћ. рЃЕрЃЋрЃћрЃюрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃљрЃерЃў, рЃћрЃА рЃўрЃЦрЃюрЃћрЃЉрЃљ рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃћрЃЉрЃўрЃА рЃўрЃАрЃбрЃЮрЃарЃўрЃўрЃА (рЃЏрЃљрЃЌрЃў рЃАрЃљрЃ«рЃћрЃџрЃћрЃЉрЃў рЃЊрЃљ рЃБрЃЏрЃљрЃдрЃџрЃћрЃАрЃў рЃЦрЃБрЃџрЃћрЃЉрЃў) XML рЃцрЃљрЃўрЃџрЃерЃў рЃерЃћрЃюрЃљрЃ«рЃЋрЃљ/рЃгрЃљрЃЎрЃўрЃЌрЃ«рЃЋрЃљ.

    рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў:
    PlayerScore (рЃЎрЃџрЃљрЃАрЃў):
    Name (string)
    HighScore (int)

    рЃърЃљрЃАрЃБрЃ«рЃўрЃАрЃЏрЃњрЃћрЃЉрЃћрЃџрЃўрЃљ XML-рЃерЃў рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃў/рЃгрЃљрЃЏрЃЮрЃАрЃљрЃдрЃћрЃЉрЃў рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃћрЃЉрЃўрЃА рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃљрЃќрЃћ.

    XmlDataService (рЃЎрЃџрЃљрЃАрЃў):

         private const string FileName = "hangman_scores.xml";
         SaveScores(List<PlayerScore> scores): рЃАрЃЎрЃЮрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃА рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА XML рЃцрЃљрЃўрЃџрЃерЃў.
         LoadScores(): рЃЎрЃўрЃЌрЃ«рЃБрЃџрЃЮрЃЉрЃА рЃЊрЃљ рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃАрЃЎрЃЮрЃарЃћрЃЉрЃўрЃА рЃАрЃўрЃљрЃА XML рЃцрЃљрЃўрЃџрЃўрЃЊрЃљрЃю.

2.  ­ЪДа HangmanGame.BusinessLogic (Business Tier)
    рЃћрЃА рЃцрЃћрЃюрЃљ рЃерЃћрЃўрЃфрЃљрЃЋрЃА рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃџрЃЮрЃњрЃўрЃЎрЃљрЃА, рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃљрЃарЃЕрЃћрЃЋрЃљрЃА, рЃљрЃАрЃЮрЃћрЃЉрЃўрЃА рЃерЃћрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃљрЃА, рЃЦрЃБрЃџрЃћрЃЉрЃўрЃА рЃЏрЃљрЃарЃЌрЃЋрЃљрЃА рЃЊрЃљ рЃњрЃљрЃЏрЃљрЃарЃ»рЃЋрЃћрЃЉрЃљ/рЃгрЃљрЃњрЃћрЃЉрЃўрЃА рЃЊрЃљрЃЊрЃњрЃћрЃюрЃљрЃА. рЃўрЃА рЃўрЃДрЃћрЃюрЃћрЃЉрЃА DataAccess-рЃА рЃАрЃЎрЃЮрЃарЃћрЃЉрЃўрЃА рЃерЃћрЃАрЃљрЃюрЃљрЃ«рЃљрЃЊ.

    рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў:
    GameEngine (рЃЎрЃџрЃљрЃАрЃў):
    private readonly List<string> \_words (рЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃа рЃњрЃљрЃюрЃАрЃљрЃќрЃдрЃЋрЃарЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃћрЃЉрЃў).
    private string \_targetWord (рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃљрЃарЃЕрЃћрЃБрЃџрЃў рЃАрЃўрЃбрЃДрЃЋрЃљ).
    private int \_guessesLeft = 6 (рЃњрЃљрЃАрЃљрЃ«рЃАрЃюрЃћрЃџрЃў рЃЏрЃфрЃЊрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў).
    private char[] _guessedWordDisplay (рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃАрЃбрЃљрЃбрЃБрЃАрЃў, рЃЏрЃљрЃњ: _ \_ \_ \_ \_).
    private List<char> \_incorrectGuesses (рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃљрЃЊ рЃюрЃљрЃгрЃўрЃюрЃљрЃАрЃгрЃљрЃарЃЏрЃћрЃбрЃДрЃЋрЃћрЃџрЃћрЃЉрЃў рЃљрЃАрЃЮрЃћрЃЉрЃў).

             рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:
                 StartNewGame(): рЃўрЃарЃЕрЃћрЃЋрЃА рЃерЃћрЃЏрЃЌрЃ«рЃЋрЃћрЃЋрЃўрЃЌ рЃАрЃўрЃбрЃДрЃЋрЃљрЃА рЃЊрЃљ рЃўрЃюрЃўрЃфрЃўрЃљрЃџрЃўрЃќрЃљрЃфрЃўрЃљрЃА рЃБрЃЎрЃћрЃЌрЃћрЃЉрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃА.

                 GuessLetter(char letter): рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃљрЃАрЃЮрЃА, рЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃА \_guessedWordDisplay-рЃА рЃЊрЃљ рЃљрЃЏрЃфрЃўрЃарЃћрЃЉрЃА \_guessesLeft-рЃА рЃЌрЃБ рЃљрЃАрЃЮ рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃўрЃљ. рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЉрЃБрЃџрЃћрЃљрЃю рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА (рЃљрЃАрЃЮ рЃАрЃгрЃЮрЃарЃўрЃљ рЃЌрЃБ рЃљрЃарЃљ).

                 CheckIfWordGuessed(): рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА, рЃљрЃарЃўрЃА рЃЌрЃБ рЃљрЃарЃљ рЃДрЃЋрЃћрЃџрЃљ рЃљрЃАрЃЮ рЃњрЃљрЃ«рЃАрЃюрЃўрЃџрЃў.

                 GetGameStatusDisplay(): рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃЋрЃўрЃќрЃБрЃљрЃџрЃБрЃа рЃАрЃбрЃљрЃбрЃБрЃАрЃА (рЃЏрЃљрЃњ: "a \_ \_ l e").

                 IsGameOver(): рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃЊрЃљрЃАрЃарЃБрЃџрЃЊрЃљ рЃЌрЃБ рЃљрЃарЃљ рЃЌрЃљрЃЏрЃљрЃерЃў (\_guessesLeft 0-рЃќрЃћ рЃюрЃљрЃЎрЃџрЃћрЃЉрЃўрЃљ рЃљрЃю рЃАрЃўрЃбрЃДрЃЋрЃљрЃљ рЃњрЃљрЃ«рЃАрЃюрЃўрЃџрЃў).

                 DidWin(): рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃЉрЃБрЃџрЃћрЃљрЃю рЃЏрЃюрЃўрЃерЃЋрЃюрЃћрЃџрЃЮрЃЉрЃљрЃА (рЃњрЃљрЃўрЃЏрЃљрЃарЃ»рЃЋрЃљ рЃЌрЃБ рЃљрЃарЃљ).

                 GetCurrentTargetWord(): рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА рЃАрЃарЃБрЃџ рЃАрЃљрЃЏрЃўрЃќрЃюрЃћ рЃАрЃўрЃбрЃДрЃЋрЃљрЃА.

         ScoreManager (рЃЎрЃџрЃљрЃАрЃў):
             private readonly XmlDataService \_dataService (рЃЊрЃљрЃЏрЃЮрЃЎрЃўрЃЊрЃћрЃЉрЃБрЃџрЃћрЃЉрЃљ Data Tier-рЃќрЃћ).

             SaveScore(string playerName, int score): рЃўрЃбрЃЋрЃўрЃарЃЌрЃћрЃЉрЃљ, рЃљрЃюрЃљрЃ«рЃџрЃћрЃЉрЃА (рЃЌрЃБ рЃљрЃ«рЃљрЃџрЃў рЃЦрЃБрЃџрЃљ рЃБрЃцрЃарЃЮ рЃЏрЃљрЃдрЃљрЃџрЃўрЃљ) рЃЊрЃљ рЃўрЃюрЃљрЃ«рЃљрЃЋрЃА рЃАрЃЎрЃЮрЃарЃћрЃЉрЃА.

             GetTopScores(int count): рЃљрЃЉрЃарЃБрЃюрЃћрЃЉрЃА TOP N рЃЏрЃЮрЃЌрЃљрЃЏрЃљрЃерЃћрЃА.
                 рЃЦрЃБрЃџрЃћрЃЉрЃў рЃњрЃљрЃЏрЃЮрЃўрЃЌрЃЋрЃљрЃџрЃЮрЃЌ рЃЏрЃљрЃњ. 100 рЃЦрЃБрЃџрЃљ - (рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃЏрЃфрЃЊрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃўрЃА рЃарЃљрЃЮрЃЊрЃћрЃюрЃЮрЃЉрЃљ \* 10).

3.  ­ЪќЦ№ИЈ HangmanGame.Presentation (Presentation Tier)
    рЃћрЃА рЃцрЃћрЃюрЃљ рЃБрЃарЃЌрЃўрЃћрЃарЃЌрЃЦрЃЏрЃћрЃЊрЃћрЃЉрЃА рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃЌрЃљрЃю (Console/WPF/Web) рЃЊрЃљ рЃўрЃДрЃћрЃюрЃћрЃЉрЃА BusinessLogic-рЃўрЃА рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃА.

рЃФрЃўрЃарЃўрЃЌрЃљрЃЊрЃў рЃЎрЃџрЃљрЃАрЃћрЃЉрЃў:
Program (рЃЌрЃБ Console рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃљрЃљ):

    private readonly GameEngine \_gameEngine
    private readonly ScoreManager \_scoreManager

рЃЏрЃћрЃЌрЃЮрЃЊрЃћрЃЉрЃў:

    Main(): рЃЏрЃЌрЃљрЃЋрЃљрЃарЃў рЃерЃћрЃАрЃарЃБрЃџрЃћрЃЉрЃўрЃА рЃгрЃћрЃарЃбрЃўрЃџрЃў.

    RunGame(): рЃЌрЃљрЃЏрЃљрЃерЃўрЃА рЃфрЃўрЃЎрЃџрЃў (рЃљрЃ«рЃљрЃџрЃў рЃЌрЃљрЃЏрЃљрЃерЃў, рЃљрЃАрЃЮрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃљ, рЃАрЃбрЃљрЃбрЃБрЃАрЃўрЃА рЃЕрЃЋрЃћрЃюрЃћрЃЉрЃљ, рЃАрЃљрЃЉрЃЮрЃџрЃЮрЃЮ рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃњрЃљрЃЏрЃЮрЃфрЃюрЃЮрЃЉрЃљ).

    DisplayWelcome(): рЃЏрЃўрЃАрЃљрЃџрЃЏрЃћрЃЉрЃљ рЃЊрЃљ рЃЏрЃћрЃюрЃўрЃБ.

    DisplayCurrentStatus(): рЃњрЃљрЃЏрЃЮрЃљрЃЦрЃЋрЃА рЃЏрЃўрЃЏрЃЊрЃўрЃюрЃљрЃарЃћ рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃАрЃбрЃљрЃбрЃБрЃАрЃў, рЃЊрЃљрЃарЃЕрЃћрЃюрЃўрЃџрЃў рЃЏрЃфрЃЊрЃћрЃџрЃЮрЃЉрЃћрЃЉрЃў рЃЊрЃљ рЃљрЃарЃљрЃАрЃгрЃЮрЃарЃў рЃљрЃАрЃЮрЃћрЃЉрЃў.

    DisplayTopScores(): рЃўрЃДрЃћрЃюрЃћрЃЉрЃА ScoreManager-рЃА TOP 10-рЃўрЃА рЃЏрЃўрЃАрЃљрЃдрЃћрЃЉрЃљрЃЊ рЃЊрЃљ рЃњрЃљрЃЏрЃЮрЃАрЃљрЃбрЃљрЃюрЃљрЃЊ.

    HandleFinalGuess(): рЃЏрЃЮрЃЏрЃ«рЃЏрЃљрЃарЃћрЃЉрЃћрЃџрЃА рЃАрЃЌрЃ«рЃЮрЃЋрЃА рЃЏрЃЌрЃџрЃўрЃљрЃюрЃў рЃАрЃўрЃбрЃДрЃЋрЃўрЃА рЃерЃћрЃДрЃЋрЃљрЃюрЃљрЃА рЃЊрЃљ рЃљрЃЏрЃЮрЃгрЃЏрЃћрЃЉрЃА рЃЏрЃљрЃА.
